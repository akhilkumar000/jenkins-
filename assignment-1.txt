BEGIN

Declare variables:
   - number (integer)
   - result (integer)

Make the user to enter a number
Read the input and store it in the variable "number"

If number is even:
   Calculate the square of the number and store the result in "result"
   Then output "The square of the number is: result"

Else if number is odd
   Calculate the cube of the number and store the result in "result"
   Output "The cube of the number is: result"

END
